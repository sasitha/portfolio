This solution was developed to be a low-cost multi-tenant SAAS application for small and medium scale businesses.

The Software was designed and developed from scratch based on the customer requirements.

A back-end REST API was developed using AWS Lambda to make the application cost-effective. An AWS DynamoDB table with a single table design (per tenant) was used to further reduce the operational cost of the software system and to achieve data separation among tenants.

The front-end application was developed as a React application.

The application infrastructure was automated with AWS CDK and the deployment was automated with the AWS CodePipeline and AWS CodeDeploy.
