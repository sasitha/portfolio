This product is intended to deliver a simple, low-cost Utility billing and reporting platform, which can support multiple Retailers. It aims to reduce manual work required by Retailers to manage their customer and reporting needs, integrating with key external systems, and automating several processes.

### Product Summary

BillBot will be targeted to support small to medium-sized electricity retailers, as well as independent organizations with a customer network that needs to be on-billed.

BillBot’s vision is to:

1. Provide a multi-tenanted platform for Retail operations
   Supporting Customer billing, reporting, and financial reconciliation

2. Be simple to use
   An intuitive, low touch system
   Easily operated without a high level of the domain or technical knowledge

3. Enable low operational costs
   Automate standardized business process
   B2B integration with supporting external systems

4. Provide a secure and scalable platform
   The system is built to industry standards and can support multiple Retailers while maintaining data isolation for each

BillBot was intended to be delivered in a phased approach so that features would be prioritized and built to meet the prioritized objectives:

#### Objective 1

Get Paua onto the BillBot platform and able to perform core functions within the system (i.e. Billing and Reporting)

#### Objective 2

Enhance the functionality to add features which will improve current practice and audit compliance – providing a voice assistant using Amazon Alexa
