This Digital Transformation project transformed a New Zeland based trailer rental company to a new generation business operation from its manual bookkeeping process. Project delivered the following outcomes to enable a unique digital transformation experience for all the stakeholders.

- A KIOSK Application for the customers to self-manage their bookings on site.
- A Web-based management portal for the staff to manage their 1000+ trailers located in 20+ partner sites.
- A Web portal for partners to insert up-to-date information about the trailers they have on their locations.

## KIOSK

KIOSK application allows the users to self-manage their bookings on site. It reduces the wait time faced by the customers as well as the on-site staff.
This application runs on top of the Android platform. That makes it is possible to run this application on a variety of Android devices. It reduces the device cost in comparison with a custom KIOSK device.

#### Technologies used

_ReactNative_

## Operations Portal

This web application allows managing trailers, bookings, and other information and all business operations. Partners will have limited access to the system. They are only allowed to see the data relevant to them.

#### Technologies used

_React_,_HTML5_,_CSS_

## Backend API

Backend API is a Springboot application, and it runs on top of the AWS Fargate service. It uses PostgreSQL as the database server.

#### Technologies used

_Springboot_,_PostgreSQL_,_AWS_
